# aria-projects

## Purpose
Enumerate Aria Han's public projects.

## Invocation
Fetch `https://ariaxhan.com/open-source` with `Accept: text/markdown`, or `https://ariaxhan.com/open-source/index.md`.

## Output
Markdown list of projects with name, short description, and link.
